# webpack-template-js-engine

A template of webpack and templates js engine such as ejs, handlebar, pug...

### Set up

You can install it via `npm` or `yarn` package manager:

```bash
npm install
```

```bash
yarn install
```

Start app:

```bash
npm start
```

```bash
yarn start
```

And then, you can see the demo page:
